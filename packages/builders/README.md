## 🚀 About

`@nyxojs/builders` is a comprehensive collection of type-safe builders for creating Discord application components. With
fluent API design and complete TypeScript support, it provides an intuitive way to build slash commands, interactive
components, rich embeds, and more while ensuring compliance with Discord's API requirements.

Perfect for creating complex slash commands, interactive UI components, rich message embeds, and any Discord application
elements that require structured data.

> [!NOTE]
> This package is part of the Nyxo.js ecosystem but can be used independently in any Discord bot or Node.js application.

## ✨ Key Features

### 🔧 **Application Command Builders**

- **Slash Commands** - Create global and guild-specific commands with ease
- **Command Options** - Support for all option types (string, integer, user, etc.)
- **Subcommands & Groups** - Build complex nested command structures
- **Autocomplete Support** - Enable dynamic option completion

### 🎮 **Interactive Component Builders**

- **Action Rows** - Container components for organizing interactive elements
- **Buttons** - Create clickable buttons with various styles and actions
- **Select Menus** - Build dropdown menus for user selections
- **Text Inputs** - Modal form inputs with validation
- **Advanced Components** - Media galleries, containers, separators, and more

### 💬 **Message Enhancement Builders**

- **Rich Embeds** - Create beautiful embedded content with images and fields
- **Color Management** - Type-safe color resolution with named constants
- **Layout Components** - Text displays, sections, and thumbnail components
- **Media Support** - File attachments and media gallery displays

### 🛡️ **Developer Experience**

- **100% TypeScript** - Complete type safety with intelligent autocompletion
- **Fluent API Design** - Method chaining for readable and maintainable code
- **Validation Built-in** - Automatic compliance with Discord's limits and requirements
- **Immutable Output** - Generate read-only objects safe for API transmission

### 🎨 **Styling & Customization**

- **Named Colors** - Predefined color constants for consistent theming
- **Layout Control** - Advanced component positioning and spacing
- **Responsive Design** - Components that adapt to different Discord clients
- **Accessibility** - Built-in support for alt text and ARIA features

## 📦 Installation
